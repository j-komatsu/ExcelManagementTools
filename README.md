# ExcelManagementTools

## 概要
ExcelManagementToolsは、Excel上でスケジュール管理や進捗確認ができるマクロツールです。タスクの進行状況や工数計算を自動化し、効率的なスケジュール管理をサポートします。

---

## 初期設定
### 1. マクロ有効化
1. Excelファイルを **`.xlsm` (マクロ有効ブック)** として保存してください。
2. Excelの「オプション」 → 「セキュリティセンター」 → 「マクロの設定」からマクロを有効にします。

### 2. VBAコードのコピー
1. 開発タブの「Visual Basic」をクリックします。
2. 「標準モジュール」を追加し、このリポジトリに含まれる `ExcelManagementTools` のコードをコピー＆ペーストしてください。
3. 保存してエディタを閉じます。

---

## 仕様
- **対象プラットフォーム**: Microsoft Excel (推奨バージョン: 2016以降)
- **主要機能**:
  - タスク進捗状況の自動更新
  - 工数（平日・祝日を考慮）自動計算
  - カレンダー形式でのタスク表示
- **対応言語**: 日本語

---

## 使い方
### 1. 進捗カレンダーの作成
1. 開発タブの「マクロ」ボタンをクリックします。
2. `CreateCalendar` を実行します。
3. 指定されたダイアログに従い、カレンダーの開始日と終了日を入力します。

### 2. タスクの追加
- `進捗カレンダー`シートの空行にタスク情報を入力します。
  - 必須項目: **タスク名**、**予定開始日**、**予定終了日**
  - 担当者名や進捗状況は任意で入力してください。

### 3. 再設定ボタンの追加
初期設定後にカレンダーの日付範囲を変更したい場合は、以下の手順で「再設定ボタン」を追加してください。

1. **ボタンの挿入**:
   1. 開発タブの「挿入」→「フォームコントロール」から「ボタン」を選択。
   2. `進捗カレンダー`シート上にボタンを配置します。

2. **マクロの割り当て**:
   1. ボタンを右クリックし「マクロの登録」を選択。
   2. マクロ一覧から `ResetTargetRangeByDate` を選択して「OK」をクリック。

3. **ボタンのラベルを変更**:
   - ボタンのテキストを「再設定」に変更。

### 4. カレンダーの日付範囲を再設定
1. 再設定ボタンをクリックします。
2. ダイアログが表示されたら、カレンダーの新しい開始日と終了日を入力します。
3. カレンダーの日付範囲が更新され、進捗状況や工数が自動で再計算されます。

---

## 注意点
1. **日付のフォーマット**:
   - 予定開始日・終了日は **YYYY/MM/DD** フォーマットで入力してください。
   - 無効な日付が入力されている場合、進捗状況が正しく反映されません。

2. **祝日の設定**:
   - 祝日は `IsHoliday` 関数に定義されています。
   - 必要に応じてカスタマイズしてください。

3. **既存データへの影響**:
   - カレンダー作成や再設定時、既存のデータが上書きされる可能性があります。
   - 重要なデータは別途バックアップを取ってください。

---

## カスタマイズ
このツールは自由にカスタマイズしてご利用いただけます。コードの変更が必要な場合は、以下の箇所を参考にしてください。

- **祝日の変更**: `IsHoliday` 関数内の祝日リストを更新。
- **カレンダーのデザイン**: `UpdateCalendarBackground` 関数内のセルスタイルを変更。

## タスク管理マクロ

### 概要
タスク管理マクロは、Excelでタスクを効率的に管理するためのツールです。タスク情報を表形式で整理し、進捗や優先度を簡単に把握できます。

### 機能一覧
| 項目       | 詳細                                                         |
|------------|--------------------------------------------------------------|
| **タスクID** | 自動連番でタスクを一意に識別。                                |
| **カテゴリ** | タスクのカテゴリを分類。                                      |
| **優先度**   | 入力規則（低、中、高、最高）で設定可能。                      |
| **ステータス**| 未着手、仕掛中、確認中、完了の中から選択可能。                |
| **期日**     | タスクの締切日を入力（YYYY/MM/DD形式）。                       |
| **開始日**   | タスクの開始日を入力（YYYY/MM/DD形式）。                       |
| **進捗率**   | 0%〜100%（10%刻み）で設定可能。                               |
| **内容**     | タスクの詳細を記載。                                          |
| **備考**     | その他のメモ。                                               |

### 使い方
1. 開発タブの「マクロ」から `CreateTaskManagementTable` を実行します。
2. 自動生成された表にタスク情報を入力します。
3. 必要に応じて進捗率やステータスを更新してください。

### 注意点
- 表外の罫線は適用されません。
- 必ずマクロ有効ブック（*.xlsm）形式で保存してください。
